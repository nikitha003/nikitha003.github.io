Intro

Hello! I'm Nikitha, a Computer Science graduate from the University of Manchester, passionate about building secure and interpretable AI systems. My work sits at the intersection of machine learning and cybersecurity, with a particular focus on detecting financial scams using hybrid NLP and ML models.

I recently completed a research-led final year project developing a real-time financial scam detection system. The architecture features a hybrid classification pipeline that combines lexical URL analysis with deep semantic inspection of webpage content, deployed via a FastAPI microservice. The goal was to enhance detection accuracy across diverse and evolving online threat vectors.
